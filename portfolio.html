Tallest Mountain (Java)

This app creates and compares instances of different landforms, specifically Mountain and Mesa, by implementing a common interface called Formation. It stores these instances in an ArrayList, determines which landform is the tallest, and prints out detailed information about each one.

Dice Roller (C#)

The application “rolls” two n-sided dice, displaying the results of each along with a total For 6-sided dice, the application recognizes the following dice combinations and displays a message for each. It should not output this for any other size of dice.

Dad's Coffee Shop (C#)

This app serves as a self-service terminal or cash register for "Dad's Coffee Shop," enabling customers to browse a menu of products, select items and quantities, view their order total, and complete the purchase using various payment methods.
